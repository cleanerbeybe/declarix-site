# Decision ledger

| Decision | Choice | Reason |
| --- | --- | --- |
| Publication format | Live HTML report plus CSV and SVG | Searchable, citeable, accessible, and directly linkable without a gate. |
| Primary schema | `Report` + `Dataset` + breadcrumbs | Matches a dated research report and its aggregate data download. No FAQPage or HowTo markup. |
| Hero emphasis | 402 qualified, 1,203 decisions, 50 priority, 397 product-fit result | Leads with the useful signal and keeps methodology out of the acquisition hero. |
| Commercial bridge | Existing CMS preserved; document assembly reduced | Directly connects the strongest rubric pattern to the current product wedge without claiming measured adoption. |
| Conversion path | Desk calculator, pack checker, registration kit, 20-minute call | Gives high-, medium-, and low-intent readers a useful next action. |
| Data access | Ungated | Improves linkability and avoids a fake or premature email workflow. |
| Newsletter | Documentation only | The “Customs Operations Brief” specification is durable, but no signup UI is published before a real provider and consent flow exist. |
| Geography | Omitted | Avoids weak normalization and small-cell disclosure while the aggregate evidence remains strong without it. |
| Research visual | CSS charts plus semantic tables; SVG press asset | Charts are direct and on-brand; tables preserve access to exact values. |
| Design lock | Existing Paper World | Maintains the high-design site language: ink/cream, ruled ledgers, Archivo display type, mono metadata, no gradients/cards/pill overload. |
| Homepage | Untouched | Eliminates homepage copy and conversion regression risk. |

## Refero design rationale

The primary reference lock is the site's existing Paper World system. The report borrows only durable data-journalism conventions: direct labeling, full denominators, visible tables, adjacent definitions, and a clear snapshot note. It does not average in a new visual system.

The memorable detail is the oversized `402` evidence field paired with ledger-strip charts. Orange marks research and unresolved evidence; green marks qualified/strong signal and conversion; ink carries the structural base.

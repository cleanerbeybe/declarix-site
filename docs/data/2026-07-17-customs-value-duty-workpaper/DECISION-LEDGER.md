# Decision ledger

| Decision | Choice | Reason |
|---|---|---|
| Primary job | Cross-foot a user-established customs value, duty, and import VAT scenario. | A useful, linkable bottom/mid-funnel asset without pretending to replace tariff or valuation work. |
| Input model | Ten required, explicit fields with no populated defaults. | Prevents implied rates or hidden factual assumptions. |
| Result hierarchy | Duties and taxes leads; valuation ledger and planning total follow. | Gives an immediate decision number while preserving the full audit trail. |
| Conversion | Result → related tools → 20-minute numbers call. | Value precedes the ask and bridges naturally into Declarix workflow economics. |
| Gating | None. | Search value, utility, links, sharing, and trust are more valuable than an email wall. |
| Persistence | Only explicit share parameters; no financial inputs in analytics. | Makes collaboration possible while keeping telemetry coarse and intentional. |
| Design lock | Existing Declarix Paper World: Archivo, IBM Plex Mono, ink/paper/field, square borders, no gradients, radius, or decorative shadow. | Maintains the premium established system and avoids template drift. |
| Memorable move | A cross-footed three-stage valuation ledger. | Makes the calculation relationship legible and visually specific to customs work. |
| Schema | `Organization`, `WebApplication`, and `BreadcrumbList`. | Truthful structured data without restricted `FAQPage` or retired `HowTo` markup. |
| Boundary placement | Beside the result, questions, and source register. | Protects accuracy without turning the acquisition hero into legal or governance copy. |
| Homepage | No source, copy, style, or markup changes. | Enforces conversion and visual regression isolation. |

## Reference lock

No live Refero MCP was available. The implementation uses the existing deployed Paper World system
as its direct product reference and the bundled Refero craft, copy, anti-slop, and visual-workflow
guidance for spacing, type, hierarchy, state, and responsive decisions.

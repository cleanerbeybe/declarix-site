# Official source register

Checked: 2026-07-17. Only current primary GOV.UK sources support technical and current claims.

| Source | What it supports |
|---|---|
| [Customs valuation handbook](https://www.gov.uk/guidance/customs-valuation) | HMRC's current valuation framework and six-method sequence. |
| [Method 1 — transaction value](https://www.gov.uk/guidance/customs-valuation/method-1-transaction-value) | Starting with price paid or payable and applying established additions or deductions, including relevant transport and insurance treatment. |
| [Customs valuation exchange rates](https://www.gov.uk/guidance/customs-valuation/exchange-rates) | HMRC monthly tables publish foreign-currency units per £1. The workpaper accepts that direction and divides the invoice-currency amount by the user-entered rate; it does not select the applicable rate. |
| [July 2026 HMRC monthly exchange-rate table](https://www.trade-tariff.service.gov.uk/exchange_rates/view/2026-7) | Directly verifies the public table heading “Currency units per £1” and the 1.1564 EUR example used in release QA. |
| [Working out the VAT value using the customs value](https://www.gov.uk/guidance/working-out-the-vat-value-using-the-customs-value-of-the-imported-goods) | Starting with customs value and adding duty, other import charges, and relevant incidental expenses to form the import VAT value. |
| [UK Trade Tariff](https://www.gov.uk/trade-tariff) | Finding commodity codes and current measures, including ad valorem, specific, or compound duty treatment and VAT rates; the workpaper does not perform a lookup and calculates ad valorem duty only. |

## Editorial boundary

The source register supports the sequence, not a universal treatment. The page consistently says
that inclusions, deductions, rates, tariff measures, evidence, and eligibility must be established
from the actual movement and current guidance.

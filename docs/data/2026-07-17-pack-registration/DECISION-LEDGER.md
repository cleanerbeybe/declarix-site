## Reference lock

Primary reference: the current Declarix Paper World static-route system.

Preserve:

- light paper field with deep ink structure;
- orange for docket/status information, not as a generic background;
- green for decisive conversion actions;
- zero-radius controls, ruled sections, large Archivo headings, mono operational labels;
- an evidence-led page rhythm rather than generic rounded cards.

Reject:

- decorative cards, gradients, indigo/violet, emoji icons, soft SaaS styling;
- a separate visual system for the free assets;
- generic legal or governance copy directly below an acquisition hero;
- changes to homepage product copy or the existing booking flow.
